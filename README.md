# Kal

Simple calendar generator

## Run

```
# clone project
git clone https://github.com/abdmmar/kal.git
cd kal

# install dependencies
pnpm install # equivalent to npm install or yarn install

# build project # equivalent to npm test or yarn test
pnpm dev
```
